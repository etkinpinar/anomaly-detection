# anomaly-detection
Anomaly detection in Apache Spark and Flink with Machine Learning Libraries

## Installation

Cloning repository

```
git clone https://github.com/etkinpinar/anomaly-detection
cd anomaly-detection
```

### Apache Spark

```
cd spark
docker-compose up -d
```

### Apache Flink

```
cd ../flink
docker-compose up -d
```

### User-interface

```
pip install matplotlib numpy flask
python3 main.py
```
